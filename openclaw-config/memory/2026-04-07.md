## 📝 注意事项

### Windows 换行符警告
```
warning: LF will be replaced by CRLF
```
- 原因：Windows 使用 CRLF，Linux/macOS 使用 LF
- 影响：不影响使用，Git 会自动处理

### CMD vs Git Bash 命令差异
| 操作 | Git Bash | CMD |
|------|----------|-----|
| 复制文件 | `cp src dest` | `copy src dest` |
| 复制目录 | `cp -r src dest` | `xcopy src dest /E /I /Y` |
| 查看目录 | `ls -la` | `dir` |

**推荐**: 用 Git Bash，命令和 Linux 一致

## 🧠 学习方法讨论

### 问题：看懂代码但自己写不出来

**原因**：
| 能力 | 跟着学 | 独立写 |
|------|--------|--------|
| 看懂 | ✅ 有解释 | ❓ 需回忆 |
| 写出来 | ✅ 有完整代码 | ❌ 需从零构建 |
| 调试 | ✅ 有人帮改 | ❌ 需自己排查 |

**解决方案**：
1. **合上代码重写** — 卡住再看答案
2. **拆分练习** — 每天练一个难度
3. **代码模板库** — 创建 `templates.py` 默写常用结构
4. **费曼技巧** — 假装教别人，边写边解释
