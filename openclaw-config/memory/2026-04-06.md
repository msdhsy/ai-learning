## 📚 学习内容

### 1. 异步编程核心 (`01_async_basic.py`)

**关键对比**：
- 串行 (requests)：3 个 URL × 1 秒 = 3 秒
- 并发 (aiohttp)：3 个 URL 同时发 = 1 秒
- 性能提升：约 3 倍

**核心概念**：
| 关键词 | 作用 |
|--------|------|
| `async def` | 定义协程函数 |
| `await` | 暂停当前协程，让事件循环做别的事 |
| `asyncio.gather()` | 并发执行多个协程 |
| `async with` | 异步上下文管理器 |

**代码要点**：
```python
# 并发执行
tasks = [fetch_async(session, url) for url in urls]
await asyncio.gather(*tasks)

# aiohttp.ClientSession 复用连接池
async with aiohttp.ClientSession() as session:
    ...
```

### 4. PostgreSQL + asyncpg (`03_postgresql_basic.py`)

**核心 API**：
| 方法 | 作用 |
|------|------|
| `asyncpg.connect()` | 创建数据库连接 |
| `conn.execute()` | 执行 SQL（DDL、UPDATE、DELETE） |
| `conn.fetchrow()` | 查询单行 |
| `conn.fetch()` | 查询多行 |
| `conn.transaction()` | 事务处理 |
| `asyncpg.create_pool()` | 连接池（生产环境） |

**关键知识点**：
- 占位符 `$1, $2, $3` 防止 SQL 注入
- `async with conn.transaction()` 自动回滚
- 连接池 `min_size=5, max_size=20`
- `asyncpg.Record` 转 `dict` 用 `dict(row)`

**完整 CRUD 示例**：
- `create_table()` - 创建表
- `insert_user()` - 插入并返回 ID
- `get_user_by_id()` - 查询单个
- `get_all_users()` - 查询全部
- `update_user_age()` - 更新
- `delete_user()` - 删除
